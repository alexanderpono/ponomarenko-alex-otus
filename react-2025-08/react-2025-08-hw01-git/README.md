# Домашнее задание 1. Основы разработки с использованием пакетных менеджеров и Git

Работа с репозиторием 

## Цель:

понять и научиться использовать пакетные менеджеры;

освоить основы работы в среде Node.js и методологию работы с пакетами;

научиться инициализировать проект, правильно писать коммиты и оформлять пулл-реквесты;

освоить базовые операции с Git, необходимые для работы в командных проектах.

## Описание/Пошаговая инструкция выполнения домашнего задания:
1. Инициализируйте новый проект с помощью npm init или yarn init.

2. Настройте основные скрипты в файле package.json (например, скрипт для запуска)

3. Инициализируйте Git-репозиторий в проекте.

4. Создайте .gitignore файл для исключения ненужных файлов (например, node_modules).

5. Напишите несколько информативных коммитов, отражающих этапы разработки.

6. Создайте репозиторий на GitHub и свяжите его с локальным репозиторием.

7. Внесите изменения в новую ветку, создайте пулл-реквест и оформите его согласно стандартам.
