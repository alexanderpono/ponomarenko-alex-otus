# Домашнее задание 7. Списки и порталы

## Цель:

освоить порталы и списки.

## Описание/Пошаговая инструкция выполнения домашнего задания:

Необходимо создать следующие компоненты

    Компонент списка товаров/операций (в зависимости от выбранного проекта). Этот компонент принимает массив данных и выводит список их компонентов товаров/операций, который мы делали ранее
    
    Сделать динамическое добавление товаров/операций в список, для этого используйте функции создания случайного товара/операции, которые вы писали в первом домашнем задании. Добавление можно сделать с помощью кнопки "показать еще", но лучше использовать IntersectionObserver
    
    Добавить в модальном окне монтирование/размонтирование модального окна внутри body. Используйте порталы. Подсмотрите реализацию модального окна в учебном репозитории.


## Критерии оценки:
    Модальное окно (1 балл) :
        (1 балл) Модальное окно корректно добавляется и исчезает из body с использованием порталов.

    Компонент списка товаров/операций (2 балла):
        (1 балл) Компонент корректно принимает массив данных и рендерит элементы.
        (1 балл) Рендеринг осуществляется без ошибок и багов.

    Динамическое добавление товаров/операций (3 балла):
        (1 балл) Кнопка "Показать еще" корректно добавляет новые элементы в список.
        (1 балл) Генерация новых товаров/операций реализована корректно.
        (1 балл) Внешний вид компонента и кнопки соответствует требованиям.

    Новые операции добавляются с помощью IntersectionObserver (4 балла):
        (1 балл) IntersectionObserver корректно добавляет новые элементы при прокрутке.
        (1 балл) Реализация производительна и не вызывает значительных задержек.
        (1 балл) Генерация новых товаров/операций реализована корректно.
        (1 балл) Внешний вид устраивает и соответствует общему дизайну.


Максимальное количество: 10 баллов
Статус "Принято" ставится при наборе 7 и более баллов

## Компетенции:
    React API
        - Создавать и использовать функциональные и классовые компоненты
        - Понимать и использовать методы для управления состоянием и эффектами
        - Использовать React.memo и React.forwardRef для оптимизации компонентов
        - Понимать и работать с менее очевидными концепциями React (провалы событий, стабильность ключей и др.)
    Принципы программирования
        - Знать и уметь использовать react паттерны в соотвествии с принципами программирования
